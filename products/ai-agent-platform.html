Capabilities

把对话、查数、知识和工具执行放进一个可控运行时。

01

多专家路由

支持自动意图路由、专家模式、agent_id 与 @ 提及直选，多专家结果可由 Synthesizer 聚合。

02

AgentScope ReAct

Assistant、ChatBI、Knowledge 执行器基于 AgentScope 与 Toolkit 闭环调度工具。

03

企业级 ChatBI

元数据注入、Few-Shot 案例、SQL 自愈、sql_plan 卡片和我的数据门户支撑自然语言查数。

04

知识增强

树形文档管理、召回测试、语义合并、RAGFlow 托管路径与引用卡片，降低知识幻觉。

05

MCP 工具生态

通过 MCP 连接企业系统、插件与生产力工具，让智能体具备可审计的业务执行能力。

06

嵌入与审计

Embed Chat SDK 对接现有鉴权和租户体系，Trace 时间线、导出与水印保障生产可控。

Workflow

智能体执行链路。

  1. 入口接入来自管理后台、Embed SDK、数据门户或业务系统的对话请求进入核心网关。
  2. 路由分发根据意图、会话上下文、专家选择和权限状态分配到 Assistant、ChatBI 或 Knowledge。
  3. 工具执行AgentScope ReAct 调用数据查询、知识检索、MCP 工具或 OpenClaw 网关。
  4. 结果合成流式返回正文、引用、SQL 计划、Trace 日志和导出结果。
  5. 运营治理通过 Prompt Factory、任务中心、审计日志和记忆管理持续优化。

Use Cases

适合把企业 AI 从聊天入口推进到真实业务执行。

企业级 ChatBI

用自然语言查业务指标，同时保留权限、元数据、SQL 护栏和可追踪结果。

知识增强问答

把非结构化文档、语义元数据和业务口径接入智能体上下文。

业务系统嵌入

通过 Embed SDK 把智能体嵌到门户、工单、运营后台或内部系统。
