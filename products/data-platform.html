Capabilities

从数据源到数据产品，把 API 做成可运营资产。

01

资源服务化

TABLE 零代码映射与 SQL 模板封装双模式，支持 Jinja2 参数分支和统一 DSL 查询入口。

02

SQL Lab

在线编写、调试、AI 辅助生成与修复 SQL，调试通过后可直接发布为生产 API 资源。

03

元数据治理

统一管理 MySQL、ClickHouse、Oracle 连接，沉淀数据集、表、字段、指标和实体关系。

04

数据产品目录

按业务域浏览 API 产品，支持权限申请、负责人审批、精选推荐和产品生命周期管理。

05

安全审计

API Key + Session 双认证，RBAC、数据脱敏、AST/SQL 静态护栏与按天分表审计日志。

06

全链路观测

24h/7d 调用趋势、Top 排行、分钟级统计聚合和连接池健康状态，支撑持续运营。

Workflow

典型数据消费流程。

  1. 注册数据源配置 MySQL、ClickHouse、Oracle 连接与角色隔离策略。
  2. 定义资源通过表映射或 SQL Lab 创建 API 资源，补充参数、描述和负责人。
  3. 发布目录把接口上架为数据产品，配置业务域、可见范围、申请审批流程。
  4. 业务消费业务用户浏览目录、申请权限，系统或 Agent 通过标准化接口调用。
  5. 审计运营通过 Trace ID、调用趋势、资产全景和零调用告警持续治理。
