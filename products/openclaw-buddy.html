OpenClaw Buddy

Chat V3 一站式 AI 工作台

为 OpenClaw 打造的零门槛 Web 管理台与对话伴侣，把聊天、Workspace 文件、Skills 技能、 Console 终端、Bot、模型和渠道管理放进同一块工作台。

Capabilities

让 OpenClaw 从命令行工具变成可运营的 Web 工作台。

01

Chat V3 工作台

多会话、流式聊天、Workspace 文件、Skills、Console、诊断面板同屏协作，减少窗口切换。

02

Workspace 文件

按 Bot 感知专属工作区，支持在线浏览、编辑、上传、附加到对话和上下文联动。

03

Console 终端

基于 WebSocket 的远程终端自动定位到当前 Bot 工作区，适合边聊边看输出。

04

Bot / Model 管理

解析 OpenClaw 配置，管理 Bot、专家模板、模型渠道、默认模型和连通性测试。

05

渠道与插件

可视化安装、启用与热重载 Skills、微信、飞书、Telegram、QQ Bot 等渠道插件。

06

守护与自愈

心跳探针、配置备份回滚、异步任务、审计大屏、系统指标和飞书报警支撑日常运营。

Workflow

从配置到对话，再到运营闭环。

  1. 接入 OpenClawBuddy 侧挂运行，读取 OpenClaw 配置目录、模型、Bot 和插件状态。
  2. 配置资产在 Web 面板创建 Bot、绑定模型、启用 Skills 和渠道插件。
  3. 进入 Chat V3在同一工作台完成聊天、文件修改、技能调用和终端输出查看。
  4. 绑定渠道通过微信、飞书、Telegram 等渠道接入真实用户会话，并按渠道归类过滤。
  5. 持续守护监控心跳、日志、设备授权和自愈事件，异常时自动恢复或报警。

Screens

Use Cases

适合把开放式 Agent 网关交给更多角色使用。

非技术人员管理 Agent

不用记 CLI，在 Web 面板完成 Bot、模型、插件和渠道配置。

Chat V3 日常工作台

把聊天、文件、技能和终端放在一起，适合持续调试和运营 Agent。

多渠道 Agent 运营

统一处理微信、飞书、Telegram 等入口的会话、授权、日志和告警。
